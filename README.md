# LSPDFR Agency Backup & Change Guide

This repository is a general guide for understanding how police agencies work in LSPDFR for GTA V single player.

It covers how agency systems are structured and how different parts like vehicles, uniforms, and names are organized inside the game files.

---

## What this is

This is not a mod pack.

This is a reference guide for:
- Understanding agency structure in LSPDFR
- Learning how agency systems are organized
- Helping users understand what different files control

---

## Requirements

- LSPDFR
- RagePluginHook
- OpenIV

---

## Important

This repository does not include any mod files.

All mods belong to their original creators and should be downloaded from official sources.
